## Local Storage

1. A developer would use local storage for a web application to save changes or history from a previous visit to the website and have them carry over to future visits. By utilzing local storage, a website will not be reset every time it is re-opened.
2. Information that should not be stored in local storage are information pertaining to security, such as passwords. Information stored local storage is not secure and can cause security issues.
3. Local storage can store string datatypes. You can convert data into a string by using the JSON.stringify() and JSON.parse() methods. These methods take data stored in an object and displays it as a string.
